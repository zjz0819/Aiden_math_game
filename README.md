# Aiden's Math Games

A lightweight browser app for turning math games into short, playful lessons.

Open `index.html` in a browser to play. No build step is required.

## First Games

- Make Ten Trail: addition pairs and complements
- Shape Workshop: definitions, properties, and shape-family relationships
- Array Builder: multiplication as rows and columns
- Share Fair: division as equal sharing
- Fact Family: multiplication and division as related facts
- Clock Quest: reading analog time
- Robot Path: Scratch-like command sequencing
- Pattern Machine: finding and extending number rules
- Quantum Tic-Tac-Toe: two-player ghost marks, loops, and collapse
- Pebble Duel: a two-player take-away strategy game
- Box Capture: a two-player square-claiming strategy game

## Reward And Difficulty System

- Correct solo rounds earn sparks.
- Winning a two-player duel earns extra sparks.
- Sparks increase Aiden's level.
- Higher levels change the difficulty of several games, such as larger addition targets, harder multiplication facts, harder division facts, harder clock times, longer patterns, and larger duel boards.
- Badges appear after meaningful milestones.

## Book-Inspired Framework

This app can use *Math Games with Bad Drawings* as a teaching framework without copying the book text. For each game you add, keep a small card:

- Game name
- Math idea
- Number of players
- Win/loss condition
- What Aiden predicts before playing
- What the app tracks or simulates
- One question for parent-child discussion
- One computational thinking link
